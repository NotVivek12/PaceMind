# PaceMind Demo Script

Target: 2 minutes. Rehearse this exact flow 3 times.

## Sample Notes

Paste this into the notes box:

```text
Photosynthesis is the process plants use to convert light energy into chemical energy. Chlorophyll absorbs light in the chloroplasts. The light-dependent reactions split water and produce oxygen, ATP, and NADPH. The Calvin cycle uses carbon dioxide, ATP, and NADPH to build glucose. Light intensity, carbon dioxide concentration, and temperature can limit the rate of photosynthesis.
```

## Talk Track

0:00-0:15 - Problem

"Most tutoring software only reacts after a student gets something wrong. PaceMind reacts earlier, using hesitation, edits, pace, and optional facial signals to spot confusion or frustration."

0:15-0:35 - Upload notes

"I will paste a short class note. The AI turns it into concepts we can quiz against immediately."

0:35-0:55 - Curriculum appears

"Now we have a learning path from raw notes: inputs, light reactions, Calvin cycle, and limiting factors."

0:55-1:20 - Answer questions

"I answer once normally. PaceMind tracks correctness, response time, backspaces, pauses, and mood state."

1:20-1:40 - Simulate frustration

"For a reliable live demo, I am using the manual mood override as the backup path. I switch to Frustrated and miss the question."

1:40-2:00 - Coach reacts

"The coach changes tone and difficulty. After ending the session, the teacher view shows mood over time, concept mastery, and a 3-sentence recap."

## Backup Paths

If webcam sensing is flaky:

1. Use the mood override dropdown.
2. Pick `Frustrated`.
3. Answer incorrectly or wait for timeout.

If AI calls are slow:

1. Continue with fallback questions.
2. Say: "The app has deterministic fallbacks so the learning loop never stalls."

If the live session breaks:

1. Return home.
2. Click `Open demo data`.
3. Show the pre-seeded dashboard.
